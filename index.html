Класс "circle-albom" и два, о которых пойдет речь ниже служат для красивой визуализации фотокарточек.
"circle-albom" позволяет организовывать изображения в виде кругового меню. Количество исспользуемых для такой организации фотокарточек ограниченно только соотношением размер\количество, так как при большом числе карточки "накладываються" одна на другую.
Навигация по фотокарточкам с выводом выбранного изображения по центру, производиться либо путем клика на миниатюру фотокарточки, либо при помощи навигационных стрелок.

Класс "chaos" Исспользует Джаваскрипт, в частности JQuery для создания случайного перемещения Фотокарточек по выделенному пространству. Вы можете остановить перемещение кликнув по пустому пространству отведенному под перемещение, а так же лучше Рассмотреть фотокарточку Кликнув по ней.
Елементы корректно отображаються как при Альбомной ориентации, так и при книжной, вне зависимости от соотношений сторон браузера

Класс "carousel" представляет собой самописный слайдер на JQuery с отключаемыми елементами упроавления, вроде меню снизу.
Перейти к следующему\предідущему изображению можно либо кликнув по их кускам слева\справо соответственно, либо выбрав нужній номер в меню под ними
