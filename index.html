Класс "circle-albom" и два, о которых пойдет речь ниже, служат для визуализации организации фотокарточек.
"Сircle-albom" позволяет организовывать изображения в виде кругового меню. Количество используемых для такой организации фотокарточек ограничено только соотношением раз мер/количество, так как при большом числе карточки "накладываются" одна на другую.
Навигация по фотокарточкам с выводом выбранного изображения по центру производится либо путем клика на миниатюру фотокарточки, либо при помощи навигационных стрелок. Ниже приведены варианты кастомизации данного меню.

Класс "chaos" использует Джаваскрипт, в частности JQuery, для создания случайного перемещения Фотокарточек по выделенному пространству. Вы можете остановить перемещение, кликнув по пустому пространству отведенному под перемещение, а также лучше рассмотреть фотокарточку, кликнув на неё.
Елементы корректно отображаются как при альбомной ориентации, так и при книжной, вне зависимости от соотношения сторон браузера

Класс "carousel" представляет собой самописный слайдер на JQuery с отключаемыми елементами управления, вроде меню снизу.
Перейти к следующему\предыдущему изображению можно либо кликнув по их частям слева\справо соответственно, либо выбрав нужный номер в меню под ними
